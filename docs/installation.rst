Installation
============


Install with ``pip``
--------------------

.. code::

    $ pip install icemac.wsgi.wsgiref


Install with ``zc.buildout``
----------------------------
Just add :mod:`icemac.wsgi.wsgiref` to the ``eggs`` setting(s) in your
buildout configuration to have it pulled in automatically::

    ...
    eggs =
        icemac.wsgi.wsgiref
    ...
